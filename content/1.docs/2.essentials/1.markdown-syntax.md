Notas boas são fáceis de ler e fáceis de encontrar. Markdown ajuda a manter estrutura sem atrito.

## Títulos

Use títulos para organizar.

```md
# Título da nota
## Seção
### Subseção
```

## Ênfase

```md
**negrito**, *itálico* e ~~tachado~~
```

## Listas

```md
- Item
- Outro item

1. Passo um
2. Passo dois
```

## Links

```md
[Kortex](/)
[Nuxt](https://nuxt.com)
```

## Dica

Use sempre um resumo no topo da nota: 2–4 linhas dizendo “o que é” e “por que importa”.
