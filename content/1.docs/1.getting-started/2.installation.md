## Requisitos

- Node.js (recomendado: versão moderna, compatível com o CI)
- pnpm

## Instalar dependências

```bash
pnpm install
```

## Rodar em desenvolvimento

```bash
pnpm dev
```

## Lint e typecheck

```bash
pnpm lint
pnpm typecheck
```

> Importante: integrações com banco/serviços externos devem acontecer **somente** no server-side, via `server/api/**`.
