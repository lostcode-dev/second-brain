## Notas

- Use notas para capturar ideias, aprendizados, referências e decisões.
- Prefira títulos claros e um parágrafo de resumo no começo.
- Use tags para facilitar o resgate depois.

## Tags e coleções

- **Tags** ajudam na busca transversal (ex.: `saúde`, `estudos`, `projeto-x`).
- **Coleções** organizam por áreas/sistemas (ex.: `Aprendizado`, `Trabalho`, `Pessoal`).

## Hábitos (monitoramento sem culpa)

Hábitos aqui não são “tudo ou nada”. O foco é consistência e tendência.

- Hábitos podem ser **binários** (feito / não feito) ou **quantitativos** (minutos, vezes, páginas).
- Use regras flexíveis (dias de descanso, metas mínimas) para evitar burnout.

## Check-in diário (Ritual)

Um check-in curto ajuda a fechar o dia e criar memória:

- O que eu foquei hoje?
- Qual hábito foi mais difícil?
- Um insight do dia

O ideal é que isso gere uma nota diária para revisão semanal.
