O **Kortex** é um sistema de conhecimento pessoal (PKM) para **capturar**, **organizar** e **conectar** informação — e então **transformar em ação**.

Construído com **Nuxt** e pensado para usar **Supabase** como base de dados (sempre via rotas server-side), ele funciona como uma extensão da mente: ideias não se perdem, elas evoluem.

## O que você encontra aqui

- **Notas** para registrar ideias e referências.
- **Tags e coleções** para dar estrutura e facilitar a busca.
- **Hábitos** como sinais de identidade e objetivos (não só checklists).
- **Check-in diário** e **revisões** para aprender com o processo.

## Um fluxo simples (e que escala)

1. **Capturar**: jogue tudo na caixa de entrada.
2. **Organizar**: mova para notas, coleções e tags.
3. **Conectar**: relacione com projetos e objetivos.
4. **Revisar**: check-in diário e revisão semanal.
