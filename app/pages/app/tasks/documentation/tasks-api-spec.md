# Tasks Module --- API Spec

Base path: /api/v1/tasks

------------------------------------------------------------------------

## Create Task

POST /tasks

{ "title": "Buy groceries", "priority": "medium" }

------------------------------------------------------------------------

## List Tasks

GET /tasks

------------------------------------------------------------------------

## Get Task

GET /tasks/:id

------------------------------------------------------------------------

## Update Task

PATCH /tasks/:id

------------------------------------------------------------------------

## Delete Task

DELETE /tasks/:id

------------------------------------------------------------------------

## Complete Task

PATCH /tasks/:id

{ "status": "completed" }

------------------------------------------------------------------------

## Create Subtask

POST /tasks/:id/subtasks

{ "title": "Buy milk" }

------------------------------------------------------------------------

## List Lists

GET /task-lists

------------------------------------------------------------------------

## Create List

POST /task-lists

{ "name": "Work" }
