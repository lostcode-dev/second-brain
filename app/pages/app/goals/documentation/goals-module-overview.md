# Goals Module --- Overview

Updated: 2026-03-04

This module manages **Goals (Metas)** that can be broken into
**sub‑tasks**, connected with **habits**, and categorized by **time
horizon** and **life areas**.

------------------------------------------------------------------------

## Core Concepts

### Goal

A long or short‑term objective the user wants to achieve.

Examples: - Run a marathon - Save 10,000€ - Read 20 books

### Subtasks

Goals can be decomposed into smaller actionable tasks.

Example:

Goal: Write a book

Subtasks: - Define outline - Write chapter 1 - Write chapter 2 - Review
manuscript

### Habits Relationship

Goals may be supported by habits.

Example:

Goal: Improve physical health

Linked habits: - Exercise daily - Drink 2L water - Sleep before midnight

### Time Categories

Goals can belong to different **time horizons**:

-   Daily
-   Weekly
-   Monthly
-   Quarterly
-   Yearly
-   Long term

### Life Areas

Goals should be categorized by **life dimension**.

Examples:

-   Personal
-   Work / Career
-   Financial
-   Health
-   Spiritual
-   Relationships
-   Learning
-   Lifestyle

------------------------------------------------------------------------

## Main User Flows

1.  Create goal
2.  Define time category (daily, monthly, yearly)
3.  Define life area
4.  Break goal into subtasks
5.  Optionally link supporting habits
6.  Track progress
7.  Review progress dashboard
