# Tasks Module --- Overview (To‑Do List)

Updated: 2026-03-04

This module manages **tasks (to‑do items)** that help users organize
daily work and personal activities.

The task system integrates well with other modules like:

-   Goals
-   Habits
-   Calendar
-   Journal

------------------------------------------------------------------------

## Core Concepts

### Task

A single actionable item.

Examples:

-   Buy groceries
-   Finish project report
-   Call mom
-   Study English

------------------------------------------------------------------------

### Task Lists

Tasks can belong to lists.

Examples:

Work\
Personal\
Shopping\
Learning

------------------------------------------------------------------------

### Task Status

Typical states:

Pending\
In Progress\
Completed\
Archived

------------------------------------------------------------------------

### Task Priority

Tasks may include priority levels:

Low\
Medium\
High\
Critical

------------------------------------------------------------------------

### Due Dates

Tasks may have:

Due date\
Scheduled date\
Deadline

------------------------------------------------------------------------

### Subtasks

Tasks may contain subtasks.

Example:

Task: Plan trip

Subtasks:

-   Choose destination
-   Buy tickets
-   Book hotel

------------------------------------------------------------------------

### Tags

Tasks may include tags for filtering.

Examples:

urgent\
health\
finance

------------------------------------------------------------------------

## Main User Flows

1 Create task 2 Assign list 3 Set priority 4 Add due date 5 Add subtasks
6 Mark task as complete 7 View tasks dashboard
